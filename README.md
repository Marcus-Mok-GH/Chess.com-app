# chess.com-app

A full-stack React chess platform featuring real-time multiplayer, AI opponents, and game analysis.

## 🛠 Workflow Rules
- **Changelog**: Always update `CHANGELOG.md` after every change or feature implementation.

## 🚀 Technical Stack
- **Frontend**: React 18, Vite 7, react-chessboard
- **Backend**: Express 5, Socket.IO
- **Database**: PostgreSQL + Drizzle ORM
- **Auth**: Neon Auth (OTP)

## 📁 Repository Structure
- `api/`: Vercel serverless entry point
- `artifacts/chess/`: Frontend React application
- `artifacts/api-server/`: Backend Express server
- `lib/`: Shared libraries (DB, API specs, Zod schemas)
